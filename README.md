## 수자 야구 게임

### 🔍 목표 : 컴퓨터가 생성한 3자리 숫자를 플레이어가 맞추는 게임

### 🚨 숫자에 대한 요구사항

- 각 자리는 1 ~ 9 사이의 숫자여야한다.
- 동일한 숫자는 사용될 수 없다. 즉, 숫자는 중복되지 않아야 한다. ex ) 333, 112, 119 불가능

### 💻 게임 결과

- 각 시도에 대해 '스트라이크'와 '볼'의 개수를 출력하여 플레이어가 숫자를 맞출 수 있도록 유도한다.

---

### 💡요구사항 정의 및 설계

- 숫자와 자리의 위치가 맞으면 스트라이크, 숫자만 맞으면 볼, 숫자와 자리가 하나도 맞지 않으면 아웃이라는 힌트를 얻어 상대방(컴퓨터)의 수를 맞추면 승리한다.

  - 예) 상대방(컴퓨터)의 수가 375일때
    - 123을 제시한 경우 : 1볼
    - 345를 제시한 경우 : 2스트라이크 0볼
    - 486을 제시한 경우 : 아웃
   
- 컴퓨터는 랜덤으로 1에서 9까지의 서로 다른 임의의 수를 3개 선택한다.
- 플레이어는 컴퓨터가 랜덤으로 생성한 숫자 3개를 예측하며 입력한다.
- 3개의 숫자를 모두 맞히면 1게임이 종료가 되고, 숫자 1을 다시 입력하면 새로운 게임이 시작된다.
- 1을 입력하면 게임이 시작되고, 2를 입력하면 게임 기록을 볼 수 있다.
- 3을 입력하면 게임이 완전히 종료가 되고, 이외의 숫자를 입력하면 올바른 숫자를 입력하라는 메시지가 뜬다.

  #### 입출력 요구 사항

  입력
    - 1부터 9사이 서로 다른 3자리의 수
    - 게임이 끝난 후 재시작/게임기록/종료를 구분하는 숫자 1,2,3
 
  출력
   - 게임 시작 문구 출력
   - 입력한 수에 따른 결과를 볼, 스트라이크 개수 or 아웃 표시
   - 실행 결과
